# TAP Map - Output Template

## Google Doc Creation

Write content as markdown to `/tmp/claude/<account>_tap_map.md`, then convert:

```bash
python3 ~/.claude/plugins/cache/claude-vibe/google-tools/*/skills/google-docs/resources/markdown_to_gdocs.py \
  --input /tmp/claude/<account>_tap_map.md \
  --title "<Account Name> - TAP Map (TAM, Architecture, Powerbase)"
```

**Tips:**
- Use `--title` for new docs, `--doc-id` to append to existing docs
- For native table updates after creation, use Python with Google Docs batchUpdate API
- `replaceAllText` works for headings/paragraphs but NOT for table cells (need index-based manipulation)
- Insert table cells in REVERSE order to avoid index drift
- Batch API requests in groups of 50 (>100 causes 400 errors)
