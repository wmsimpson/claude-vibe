# Databricks Troubleshooting

Systematic troubleshooting guide for diagnosing and resolving common Databricks issues including cluster problems, job failures, authentication errors, performance degradation, and networking issues.

## How to Invoke

### Slash Command

```
/databricks-troubleshooting
```

### Example Prompts

```
"Help me troubleshoot why my customer's Databricks cluster keeps failing to start"
"Diagnose authentication errors in a customer's workspace"
"My customer's jobs are failing intermittently, help me figure out why"
```

## Prerequisites

| Requirement | Details |
|-------------|---------|
| Workspace Access | Access to the Databricks workspace experiencing issues |
| Logs/Metrics | Access to cluster logs, job logs, or error messages |

## What This Skill Does

1. Gathers information about the issue and environment
2. Categorizes the error type (cluster, job, auth, performance, networking)
3. Walks through a systematic diagnostic process
4. Recommends targeted solutions based on findings
5. Documents the resolution for future reference

## Related Skills

- `/performance-tuning` - For performance-specific issues (slow queries, high costs, spill/skew)
- `/support-escalation` - For escalating critical issues that cannot be resolved directly
