# Databricks App Templates

Reference templates from the official Databricks app-templates repository. Each template provides a starting point for building specific types of Databricks Apps.

## Usage Guide

### By Framework
- **Streamlit:** Best for data science apps, quick prototypes, dashboards
- **Dash/Plotly:** Best for complex interactive visualizations
- **Gradio:** Best for ML model interfaces, demo applications
- **Shiny:** Best for R users, statistical applications
- **Flask:** Best for RESTful APIs, microservices
- **Next.js/TypeScript:** Best for production web applications
- **MCP Servers:** Best for model context protocol implementations

### By Use Case
- **Hello World:** Start with framework-specific hello-world templates
- **Data Analytics:** Use `*-data-app` templates
- **User Authentication:** Use `*-obo-user` templates for user-scoped access
- **Chatbots:** Use `*-chatbot-app` templates
- **Database Apps:** Use `*-database-app` templates
- **Full Stack:** Use `nodejs-fastapi-hello-world-app`
- **Production Chatbots:** Use `e2e-chatbot-app` or `e2e-chatbot-app-next`

### Initialization
Templates can be used with `databricks bundle init` or cloned directly from the repository for customization.
